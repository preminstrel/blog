    Pytorch环境搭建

    Install Anaconda3

    最近在用WSL，因为比双系统方便且想直接在WSL里面写代码，于是就在WSL环境下进行了配置。首先安装Anaconda，这里是直接从官网上下package的。

    1
2

    cd ~/Downloads
bash Anaconda3-5.3.0-Linux-x86_64.sh

    这个时候我发现

    1
2

    $ conda info
zsh: command not found: conda

    这是zsh和路径的原因，我们需要添加路径

    1
2

    vim ~/.zshrc
source ~/.zshrc

    Create Environment

    1
2
3

    cd ~/anaconda3
conda create -n pytorch python=3.7
conda init zsh

    我们需要查看CUDA的类型，所以安装lightdm

    1
2

    sudo apt-get install lightdm
nvidia-smi

    可以看见，我的是CUDA Version: 11.4，于是我们在pytorch官网上找到一个CUDA 10.2版本的进行安装。

    1
2

    conda activate pytorch
conda install pytorch torchvision torchaudio cudatoolkit=10.2 -c pytorch

    Run

    1
2
3
4

    conda activate pytorch
conda install ipython 
conda install jupyter
jupyter notebook --no-browser #Mention here

    Tensorflow+Keras

    tensorflow的安装变得比以前简单了，以前keras的版本会和tensorflow有冲突，现在没了，很不错。值得注意的是，需要安装cudatoolkit，我的显卡是CUDA Version: 11.4，所以找了11.0的安装。

    1
2
3

    pip install --upgrade pip
pip install tensorflow
conda install cudatoolkit=11.0
